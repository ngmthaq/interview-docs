# SPA là gì? Ưu / nhược điểm?

- CSR
- SEO
- Performance

## Chi tiết câu trả lời

Single Page Application.

### SPA là gì?

- Ứng dụng web load một page, update dynamically.
- Client-side routing, no full page reloads.

### Ưu điểm

- Fast navigation, smooth UX.
- Less server load, API-based.

### Nhược điểm

- **SEO**: Hard cho search engines (fix với SSR).
- **Performance**: Initial load lớn, cần optimize.
- **Browser history**: Manual management.

Modern SPAs use SSR/SSG để mitigate issues.
