# Khi nào không nên dùng reactive state?

- Derived state
- Computed thay thế
- Tránh over-reactivity

## Chi tiết câu trả lời

When not to use reactive state.

### Derived state

- Use computed.
- No separate reactive.

### Computed thay thế

- Efficient, cached.

### Tránh over-reactivity

- Performance.
- Minimize reactive objects.

Use judiciously.
