# User đổi filter liên tục → gọi API nhiều

- Debounce
- Cancel request
- Cache kết quả

## Chi tiết câu trả lời

Frequent filter changes.

### Debounce

- Delay API calls.
- Wait user stop typing.

### Cancel request

- Abort old requests.
- Latest wins.

### Cache kết quả

- Store previous results.
- Avoid duplicate calls.

Optimize API usage.
