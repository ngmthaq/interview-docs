# Logic dùng chung cho nhiều component

- Custom composable (`useXxx`)
- So sánh với mixin (anti-pattern)

## Chi tiết câu trả lời

Shared logic.

### Custom composable (`useXxx`)

- Extract logic to function.
- Reusable, testable.

### So sánh với mixin (anti-pattern)

- Mixin: naming conflicts.
- Composable: explicit, flexible.

Composables preferred.
