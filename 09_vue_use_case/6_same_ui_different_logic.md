# Cùng UI nhưng logic khác nhau

- Props + strategy
- Composable
- Slot

## Chi tiết câu trả lời

Same UI, different logic.

### Props + strategy

- Pass logic via props.
- Strategy pattern.

### Composable

- Inject logic via composable.
- Flexible.

### Slot

- Pass logic via slots.
- Composition over inheritance.

Choose based on complexity.
