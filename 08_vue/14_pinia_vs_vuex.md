# Pinia vs Vuex

- Pinia: đơn giản, type-safe
- Vuex: legacy
- Khi nào chỉ cần local state?

## Chi tiết câu trả lời

State management libraries.

### Pinia

- Official Vue 3 store.
- Simple API, TypeScript support.

### Vuex

- Vue 2 legacy.
- More boilerplate.

### Khi nào chỉ cần local state?

- Small app, no complex state.
- Component-level state đủ.

Pinia recommended cho new projects.
