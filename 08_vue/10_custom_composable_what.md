# Custom composable là gì?

- Hàm `useXxx`
- Không render UI
- Chia sẻ logic

## Chi tiết câu trả lời

Custom composables.

### Hàm `useXxx`

- Function return reactive state/logic.
- Convention: start with "use".

### Không render UI

- Only logic, no JSX.
- Components use composables.

### Chia sẻ logic

- Reusable across components.
- Extract common patterns.

Composables powerful cho code reuse.
