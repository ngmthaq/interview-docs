# Validate form realtime nhưng không làm UI lag

- Debounce validate
- Validate theo field
- Async validation

## Chi tiết câu trả lời

Realtime validation without lag.

### Debounce validate

- Debounce validation calls.
- Wait user stop typing.

### Validate theo field

- Validate field individually.
- Not whole form mỗi change.

### Async validation

- Server-side validation.
- Handle async errors.

Balance UX và performance.
