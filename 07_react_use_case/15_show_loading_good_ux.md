# Hiển thị loading đúng UX

- Skeleton
- Disable action
- Optimistic UI

## Chi tiết câu trả lời

Good loading UX.

### Skeleton

- Show placeholder structure.
- Better than spinner.

### Disable action

- Disable buttons during loading.
- Prevent double submits.

### Optimistic UI

- Update UI immediately.
- Revert nếu fail.

Improve perceived performance.
