## Chi tiết câu trả lời

Simplify component APIs.

### Group props

- Group related props into object.
- Destructure in component.

### Children / render props

- Pass children as function.
- Flexible composition.

### Context nội bộ

- Internal context share props.
- Reduce prop drilling.

Improve maintainability.
