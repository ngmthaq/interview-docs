# Sync state giữa nhiều tab

- localStorage event
- BroadcastChannel
- Backend sync

## Chi tiết câu trả lời

Sync state across tabs.

### localStorage event

- Listen 'storage' event.
- Update state from other tabs.

### BroadcastChannel

- Modern API for tab communication.
- Send/receive messages.

### Backend sync

- Persist to server.
- Real-time sync via WebSocket.

Choose based on needs.
