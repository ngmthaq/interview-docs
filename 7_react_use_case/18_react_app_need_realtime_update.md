# React app cần realtime update

- WebSocket
- SSE
- Polling

## Chi tiết câu trả lời

Real-time updates.

### WebSocket

- Bidirectional communication.
- Low latency.

### SSE

- Server-sent events.
- Simple, unidirectional.

### Polling

- Periodic requests.
- Fallback nếu no WebSocket.

WebSocket best cho interactive apps.
