# Vue.js là gì? Ưu điểm chính?

- Progressive framework
- Reactive data binding
- Dễ tiếp cận, dễ tích hợp dần

## Chi tiết câu trả lời

Vue.js là framework JavaScript để xây dựng giao diện người dùng.

### Progressive framework

- Có thể dùng ít hoặc nhiều.
- Từ library đơn giản đến full framework.
- Tích hợp dần vào project existing.

### Reactive data binding

- Data binding tự động.
- Thay đổi data → UI update.
- Declarative, dễ hiểu.

### Dễ tiếp cận, dễ tích hợp dần

- Learning curve thấp.
- HTML, CSS, JS quen thuộc.
- Ecosystem phong phú.

Vue linh hoạt, scalable.
