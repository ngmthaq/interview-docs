# Composition API giải quyết vấn đề gì?

- Logic reuse
- Tách concern theo chức năng
- Thay thế mixin

## Chi tiết câu trả lời

Composition API benefits.

### Logic reuse

- Extract logic to composables.
- Share across components.

### Tách concern theo chức năng

- Group related logic together.
- Better organization.

### Thay thế mixin

- Mixin có naming conflicts.
- Composition explicit, type-safe.

Composition API powerful cho large apps.
