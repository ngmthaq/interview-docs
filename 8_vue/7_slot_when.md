# Slot dùng khi nào?

- Reusable UI
- Scoped slot
- Tránh prop drilling

## Chi tiết câu trả lời

Slot trong Vue.

### Reusable UI

- Component flexible content.
- Cha pass content vào slot.

### Scoped slot

- Pass data từ component ra slot.
- Slot access component data.

### Tránh prop drilling

- Slot cho composition.
- Không cần pass props qua levels.

Slot enable flexible components.
