# Vite khác Webpack thế nào?

- Dev nhanh
- ES module native
- Build tối ưu

## Chi tiết câu trả lời

Vite vs Webpack.

### Dev nhanh

- Native ES modules.
- No bundling in dev.

### ES module native

- Browser support ES modules.
- Fast HMR.

### Build tối ưu

- Rollup-based production build.
- Tree-shaking, code splitting.

Vite modern, fast alternative.
