# Giữ state khi chuyển route

- `keep-alive`
- Global store
- URL query

## Chi tiết câu trả lời

Persist state across routes.

### `keep-alive`

- Cache component instances.
- State preserved on navigation.

### Global store

- State in Pinia/Vuex.
- Survive route changes.

### URL query

- State in URL params.
- Shareable, bookmarkable.

Choose based on persistence needs.
