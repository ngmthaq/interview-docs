# Interceptor là gì?

- Chặn request / response
- Dùng cho:
  - Gắn token
  - Logging
  - Handle error chung

## Chi tiết câu trả lời

Axios interceptors.

### Chặn request / response

- Modify before send/receive.

### Dùng cho

- Attach auth token.
- Log requests.
- Global error handling.

Powerful for cross-cutting concerns.
