## Chi tiết câu trả lời

Axios là gì?

### Axios là HTTP client dựa trên Promise

- Promise-based API.
- Async/await friendly.

### Tự parse JSON

- Auto parse JSON response.
- No manual JSON.parse().

### Hỗ trợ interceptor, cancel request

- Request/response interceptors.
- Cancel via AbortController.

### Hoạt động tốt cả browser & Node.js

- Universal library.
- Consistent API.

Axios powerful hơn fetch.
