# Axios trả response như thế nào?

- `data`
- `status`
- `headers`
- `config`

## Chi tiết câu trả lời

Axios response object.

### `data`

- Parsed response data.

### `status`

- HTTP status code.

### `headers`

- Response headers.

### `config`

- Original request config.

Access all response info.
